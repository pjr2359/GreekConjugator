USAGE

Using SQLite3

sqlite3 dict.db < dict.sql

or

sqlite3 dict.db

and then

.read dict.sql

SQLite3 installation

debian/ubuntu
apt install sqlite3

Other DBMS

For a GUI you can use sqlitebrowser

For any other SQL DBMS (PostgreSQL, MySQL) use the appropriate commands for the import
